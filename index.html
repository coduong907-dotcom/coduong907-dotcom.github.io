项目经历

项目一

重点项目

手机壳跨境电商宣传设计

围绕手机壳产品在跨境电商平台的销售场景，完成产品图片优化、商品卖点提炼、主图设计、详情页排版和基础上架信息整理。

项目背景

手机壳类目视觉竞争强，买家需要快速理解外观、材质、兼容型号、颜色和保护卖点。

我的任务

  • 产品主图设计、商品白底图、卖点图
  • 详情页长图、产品参数展示、英文卖点文案
  • SKU / 颜色 / 型号 / 价格信息整理

最终产出

  • 一套跨境电商商品展示图片结构
  • 可用于上架前整理的 SKU 与价格信息表
  • 面向英文平台的基础卖点文案

项目二

跨境电商产品上架模拟案例

模拟 Amazon / Shopee / TikTok Shop 等跨境平台的商品上架流程，完成产品信息整理、关键词提炼、SKU 表格制作和商品展示资料整理。

项目背景

产品上架需要信息准确、字段完整，标题、卖点、分类、库存和价格需要统一整理。

我的任务

  • 产品标题整理、英文 Listing 标题
  • 5 条产品卖点 Bullet Points、产品关键词
  • SKU 表格、价格信息、商品分类、库存信息

最终产出

  • 一份商品上架信息表
  • 一组可复用的上架检查表
  • 基础英文 Listing 文案结构

项目三

电商实习工作记录

实习期间参与深圳市报喜通达电子科技有限公司日常电商业务支持，协助完成商品图片处理、价格表整理、发货信息核对及沟通对接等工作。

项目背景

公司日常电商业务需要稳定处理商品资料、图片、价格表和发货信息，强调准确性与协作效率。

我的任务

  • 商品修图前后对比、商品价格表排版
  • 产品信息整理、发货流程辅助
  • 客户 / 内部沟通对接、商品资料整理流程

最终产出

  • 更清晰的商品图和价格表资料
  • 发货核对与沟通记录整理
  • 日常电商支持工作流程经验

项目四

POP MART 大广赛视觉设计

围绕 POP MART 品牌年轻化、潮流化的视觉风格，完成系列宣传海报设计，提升品牌传播氛围和视觉吸引力。

项目背景

潮玩品牌需要鲜明的视觉记忆点，适合在海报和社交媒体场景中形成快速传播。

我的任务

  • 海报设计、品牌视觉风格梳理
  • IP 形象传播方向提炼
  • 社交媒体视觉延展

最终产出

  • 系列宣传海报版式
  • 统一的色彩和图形风格
  • 适合社媒传播的视觉延展稿

项目五

书籍装帧设计

通过书籍装帧项目展示图文排版、视觉统一、字体层级和版式设计能力，这些能力可以迁移到电商详情页、商品资料排版和宣传物料设计中。

项目背景

书籍设计强调信息层级、阅读节奏和画面统一，能够体现扎实的版式设计基础。

我的任务

  • 书籍封面、内页排版
  • 字体设计、色彩系统
  • 图文编排与整体视觉统一

最终产出

  • 完整书籍封面和内页展示
  • 清晰的字体层级与版面系统
  • 可迁移到电商详情页的排版能力

About

关于我
